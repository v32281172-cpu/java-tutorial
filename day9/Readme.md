## Binary Search

Binary Search is an efficient searching algorithm used on sorted arrays. It repeatedly divides the search space into two halves, reducing the time complexity from **O(n)** to **O(log n)**.

The following problems in this repository use the **Binary Search** algorithm:

* **35. Search Insert Position**

  * Finds the index of a target element in a sorted array.
  * If the target does not exist, returns the position where it should be inserted.
  * **Time Complexity:** O(log n)

* **34. Find First and Last Position of Element in Sorted Array**

  * Uses Binary Search twice to locate the first and last occurrence of a target element.
  * **Time Complexity:** O(log n)
