
# Day 5 - Java Number Programs and Functions

On Day 5, I focused on solving number-based programming problems in Java using loops, conditional statements, and methods. These programs helped me strengthen my understanding of digit manipulation and improve my problem-solving skills.

### Armstrong Number

Implemented a program to determine whether a given number is an Armstrong number by extracting each digit, raising it to the appropriate power, summing the results, and comparing the sum with the original number.

### Strong Number

Developed a program to check whether a number is a Strong Number by calculating the factorial of each digit and verifying if their sum equals the original number.

### Largest Digit in a Number

Created a program that extracts every digit using the modulo operator and identifies the largest digit by comparing each extracted value.

### Smallest Digit in a Number

Built a program to find the smallest digit in a given number by traversing through all digits and updating the minimum value whenever a smaller digit is found.

### Sum of Even and Odd Digits

Implemented a program that separates even and odd digits from a number and calculates their sums independently.

### Functions (Methods)

Practiced writing user-defined methods to organize code into reusable blocks. Learned how to define methods, pass arguments, return values, and call methods from the `main()` function.

## Concepts Learned

* Using the `Scanner` class for user input.
* Extracting digits using `%` (modulo) and `/` (division).
* Looping through digits using `while` loops.
* Applying conditional statements for decision-making.
* Creating reusable code using methods.
* Improving program readability through modular programming.
