# Java String Programs

## Overview

This repository contains two Java programs that demonstrate fundamental string manipulation techniques. The programs cover common operations performed using the `String` and `StringBuilder` classes, as well as reversing a string using the Two-Pointer technique.

---

## Programming Language

- Java
- JDK 8+

---

## Programs Included

### 1. String Operations

This program demonstrates various built-in methods available in the `String` and `StringBuilder` classes.

#### Operations Performed

- Creating String objects
- Converting String using `toString()`
- Comparing strings using `equals()`
- Replacing characters using `replace()`
- Replacing words using `replaceAll()`
- Checking substring using `contains()`
- Splitting a string using `split()`
- Appending text using `append()`
- Inserting values using `insert()`
- Replacing characters in `StringBuilder` using `replace()`

#### Concepts Used

- String Class
- StringBuilder Class
- String Manipulation
- Character Replacement
- String Comparison
- String Splitting

---

### 2. String Reverse

This program reverses a string by converting it into a character array and swapping characters from both ends using the Two-Pointer technique.

#### Operations Performed

- Convert String to Character Array
- Reverse characters using swapping
- Convert Character Array back to String
- Compare strings using `equals()`

#### Concepts Used

- String
- Character Array
- Two-Pointer Technique
- Swapping
- Loops
- String Comparison

---

## Algorithms Used

- Two-Pointer Technique
- String Manipulation
- Character Swapping
