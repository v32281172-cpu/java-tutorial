# Day 7 - Java Arrays

## Topics Covered
- Array declaration and initialization
- Traversing arrays using loops
- Finding maximum and minimum values
- Sum of even and odd elements
- Sum of even and odd digits
- Matrix transpose
- Nested loops

## Programs

### 1. Array Declaration and Initialization
**File:** `arraydeclarationandinitializing.java`

- Declares an array
- Initializes values
- Displays array elements

---

### 2. Addition Table
**File:** `additiontable.java`

- Generates an addition table using nested loops.
- Demonstrates loop control and formatted output.

---

### 3. Sum of Even and Odd Digits
**File:** `evenodddigitsum.java`

- Reads a number.
- Separately calculates:
  - Sum of even digits
  - Sum of odd digits

Example:
```
Input : 482915
Even Sum = 14
Odd Sum = 15
```

---

### 4. Sum of Even and Odd Numbers in an Array
**File:** `evensumoddsum.java`

- Traverses an array.
- Computes:
  - Sum of even elements
  - Sum of odd elements

---

### 5. Maximum and Minimum Value
**File:** `maxvalueminvalue.java`

- Finds the largest element.
- Finds the smallest element.
- Uses simple array traversal.

---

### 6. Matrix Transpose
**File:** `transpose.java`

- Creates a 2×3 matrix.
- Computes its transpose.
- Prints the resulting 3×2 matrix.

Example:

Original Matrix
```
1 2 3
4 5 6
```

Transpose
```
1 4
2 5
3 6
```
